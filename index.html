SkyCast

SkyCast is a weather forecast app created using the Dark Sky API and the Google Maps, Google Geolocation, and Google Geocoding APIs. To render the charts I used the ng2-charts library. Using the app you can get today, tomorrow, and next week's forecast either for your location or a searched location, or you can do a search of historic data to see the weather for a specific place and date.

Check it out! | GitHub

Maybe Spring Rolls

Maybe Spring Rolls is an application that decides where to eat for you. The goal of the project was to solve the problem of indecisiveness at dinner time. You can set parameters such as the distance you're willing to travel and the type of food you're in the mood for and it will randomly choose a location using the Google Places API. You can also log in and store a list of meals you can make at home so that in case you've been too picky it will choose something you can make at home.

Check it out! | GitHub

Cow Chucker

Cow Chucker is a game about shooting cows at Coding Dojo instructors. For this project I worked with a team and was responsible for cannon mechanics, cameras and general direction.

Controls:
Arrow Keys: Aim
Ctrl: Press to build thrust, release to shoot
C: Return to cannon

Give it a try! Windows 32-bit | Windows 64-bit | Mac OSX

Lights Out

Lights Out is a puzzle game where the goal is to turn all of the lights off, or in this version turn all red squares into blue squares. Clicking on any square will change the color of the clicked square and all orthogonally adjacent squares. The puzzles are randomly generated and should always be solvable. I referenced this page for a lot of the information that went into creating this application.
